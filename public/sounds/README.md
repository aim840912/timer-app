# 鬧鈴音效檔案

此目錄存放鬧鈴音效檔案。

## 需要的音效檔案

請將以下音效檔案放入此目錄：

1. **alarm-default.mp3** - 預設鬧鈴音效（清晰、明亮）
2. **alarm-gentle.mp3** - 溫和音效（柔和、漸強）
3. **alarm-classic.mp3** - 經典音效（傳統鬧鐘聲）

## 取得免費音效

### 推薦來源

1. **Freesound.org**
   - 免費音效資料庫
   - 需註冊帳號
   - 搜尋關鍵字：「alarm clock」、「beep」、「bell」

2. **Zapsplat.com**
   - 免費帳號可下載
   - 高品質音效
   - 搜尋：「alarm」

3. **Pixabay Sound Effects**
   - 完全免費，無需註冊
   - https://pixabay.com/sound-effects/search/alarm/

### 音效規格建議

- **格式**：MP3
- **採樣率**：44.1kHz 或更高
- **長度**：3-10 秒（會循環播放）
- **音量**：建議不要過大，避免嚇到使用者

### 臨時測試方案

如果目前沒有音效檔案，可以：

1. 使用線上音效產生器（例：https://www.tones.app/）
2. 錄製自己的聲音或音樂
3. 從手機鬧鈴音效複製（確保版權許可）
