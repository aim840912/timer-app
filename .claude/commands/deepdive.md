# Deep Dive Code Exploration

deep dive 請對以下主題進行深度探索和分析：

$ARGUMENTS

## 探索目標：

1. **程式碼搜尋** - 使用多種策略找到相關程式碼：
   - 關鍵字搜尋（函數名、類別名、變數名）
   - 檔案路徑模式搜尋
   - 依賴關係分析

2. **架構理解** - 繪製系統架構圖：
   - 資料流向
   - 模組依賴關係
   - API 端點結構
   - 資料庫 schema 關聯

3. **模式識別** - 找出現有的開發模式：
   - 錯誤處理模式
   - 認證授權流程
   - 資料驗證方式
   - 測試模式

4. **問題分析** - 識別潛在問題：
   - 程式碼重複
   - 效能瓶頸
   - 安全漏洞
   - 技術債

## 輸出格式：

- 使用 Markdown 清單和程式碼區塊
- 提供檔案路徑和行號引用（格式：`file:line`）
- 包含具體的程式碼片段
- 總結發現和建議

**提示**：請至少進行 5 次工具調用，確保探索的完整性。
